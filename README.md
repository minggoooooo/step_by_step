## 📒 프로젝트 개요
- 이 프로젝트는 자바와 Spring Framework를 활용한 웹 개발자 과정을 진행하면서 만든 팀 프로젝트입니다
- Spring Framework 수업 도중 진행 된 프로젝트라 Spring 활용을 못하고 jsp를 활용 한 프로젝트입니다.
  


## 📘 프로젝트 요약
- 내가 원하는 곳에서, 내가 원하는 사람들과 함께 산책하자! 라는 메세지를 기반으로 만든 모임위주 산책 사이트입니다
